## Task 1.1.C — `add_signer_view`

**What it does:** Appends an `EncryptedViewKey` entry to the `ViewKeyDistribution` for the cofre, growing the account via realloc. Squads-gated. Rejects duplicate signers.

**Files:**
- Create: `programs/cloak-gatekeeper/src/instructions/add_signer_view.rs`

- [ ] **Step 1: Create the file with the full content below**

`programs/cloak-gatekeeper/src/instructions/add_signer_view.rs`:

```rust
use anchor_lang::prelude::*;

use crate::errors::CloakSquadsError;
use crate::state::*;
use crate::utils::verify_squads_vault_signer;

pub fn handler(
    ctx: Context<AddSignerView>,
    signer: Pubkey,
    ephemeral_pk: [u8; 32],
    nonce: [u8; 24],
    ciphertext: [u8; 48],
) -> Result<()> {
    verify_squads_vault_signer(
        &ctx.accounts.cofre.multisig,
        0,
        &ctx.accounts.squads_vault,
    )?;

    let dist = &mut ctx.accounts.view_distribution;
    require!(
        !dist.entries.iter().any(|entry| entry.signer == signer),
        CloakSquadsError::SignerAlreadyExists
    );

    dist.entries.push(EncryptedViewKey {
        signer,
        ephemeral_pk,
        nonce,
        ciphertext,
        added_at: Clock::get()?.unix_timestamp,
    });

    Ok(())
}

#[derive(Accounts)]
pub struct AddSignerView<'info> {
    #[account(seeds = [b"cofre", cofre.multisig.as_ref()], bump = cofre.bump)]
    pub cofre: Account<'info, Cofre>,
    pub squads_vault: Signer<'info>,
    #[account(
        mut,
        seeds = [b"view_dist", cofre.key().as_ref()],
        bump = view_distribution.bump,
        realloc = ViewKeyDistribution::space(view_distribution.entries.len() + 1),
        realloc::payer = payer,
        realloc::zero = false,
    )]
    pub view_distribution: Account<'info, ViewKeyDistribution>,
    #[account(mut)]
    pub payer: Signer<'info>,
    pub system_program: Program<'info, System>,
}
```

## Task 1.1.D — `remove_signer_view`

**What it does:** Removes the entry whose `signer == target` from the `ViewKeyDistribution` and shrinks the account. Squads-gated. Rejects if signer not present.

**Files:**
- Create: `programs/cloak-gatekeeper/src/instructions/remove_signer_view.rs`

- [ ] **Step 1: Create the file with the full content below**

`programs/cloak-gatekeeper/src/instructions/remove_signer_view.rs`:

```rust
use anchor_lang::prelude::*;

use crate::errors::CloakSquadsError;
use crate::state::*;
use crate::utils::verify_squads_vault_signer;

pub fn handler(ctx: Context<RemoveSignerView>, target: Pubkey) -> Result<()> {
    verify_squads_vault_signer(
        &ctx.accounts.cofre.multisig,
        0,
        &ctx.accounts.squads_vault,
    )?;

    let dist = &mut ctx.accounts.view_distribution;
    let position = dist
        .entries
        .iter()
        .position(|entry| entry.signer == target)
        .ok_or(CloakSquadsError::SignerNotFound)?;
    dist.entries.remove(position);

    Ok(())
}

#[derive(Accounts)]
#[instruction(target: Pubkey)]
pub struct RemoveSignerView<'info> {
    #[account(seeds = [b"cofre", cofre.multisig.as_ref()], bump = cofre.bump)]
    pub cofre: Account<'info, Cofre>,
    pub squads_vault: Signer<'info>,
    #[account(
        mut,
        seeds = [b"view_dist", cofre.key().as_ref()],
        bump = view_distribution.bump,
        realloc = ViewKeyDistribution::space(view_distribution.entries.len().saturating_sub(1)),
        realloc::payer = payer,
        realloc::zero = false,
    )]
    pub view_distribution: Account<'info, ViewKeyDistribution>,
    #[account(mut)]
    pub payer: Signer<'info>,
    pub system_program: Program<'info, System>,
}
```

**Note on shrink:** Anchor's `realloc` with `realloc::zero = false` accepts both grow and shrink. On shrink, lamports are refunded to `payer`. The `saturating_sub(1)` is defensive — if `entries.len()` is 0 the realloc is a no-op against the same size, but `entries.remove(position)` would have already errored on the bounds check before we reach realloc; the `SignerNotFound` check makes this unreachable in practice.

```rust
use anchor_lang::prelude::*;

use crate::errors::CloakSquadsError;
use crate::state::*;
use crate::utils::verify_squads_vault_signer;

pub fn handler(ctx: Context<RevokeAudit>, diversifier_trunc: [u8; 16]) -> Result<()> {
    verify_squads_vault_signer(
        &ctx.accounts.cofre.multisig,
        0,
        &ctx.accounts.squads_vault,
    )?;

    let cofre = &mut ctx.accounts.cofre;
    require!(
        cofre.revoked_audit.len() < Cofre::MAX_REVOKED,
        CloakSquadsError::RevocationCapacity
    );
    require!(
        !cofre.revoked_audit.iter().any(|d| d == &diversifier_trunc),
        CloakSquadsError::RevocationCollision

        mut,

        bump = cofre.bump,
        realloc = Cofre::space(cofre.revoked_audit.len() + 1),
        realloc::payer = payer,
        realloc::zero = false,
```

```rust
pub fn handler(ctx: Context<SetOperator>, new_operator: Pubkey) -> Result<()> {
```
